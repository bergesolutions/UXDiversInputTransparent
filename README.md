A sample app to reproduce bug https://github.com/UXDivers/uxd-popups/issues/35
- Just run the app
- On the Main page, click on the Button "Show Toast with InputTransparent False"
- Notice that the "Increase Count" Button does not work anymore.
- Close the Toast
- Increase Count works again
- Click on "Show Toast with InputTransparent True"
- Increase Count still works, but the Toast cannot be closed anymore.
